## 블로그 기능 정리
- 카카오톡 로그인 기능
- 포스트, 댓글 작성
  - 이미지, 썸네일 첨부
  - 대댓글 기능
  - 비밀글, 비밀댓글 기능
  - 좋아요 (하트) 기능
- 내 정보관리 기능 (SNS 사용자 미서비스)
    - 썸네일 변경, 등록
    - 이메일, 패스워드 인증 및 변경
- SNS 공유 기능
- 방명록 기능
- 글 검색 기능
- 타사이트 검색 등록
  - 구글 검색 등록완료
  - 네이버 검색 등록완료

   - 방문자 수 통계(년, 월, 일)
 - 카테고리 관리
 - 작성 글 조회
 - 이슈 관리 기능
 - 관리자 정보 기능
